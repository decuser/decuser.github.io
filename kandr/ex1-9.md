# Example 1-9
Write a program to copy its input to its output, replacing each string of one or more blanks by a single blank.

```
vi ex1-9.c
/* ex1-9 in to out, compress blanks
 * created 20180224.1659 */
#include <stdio.h>

int main()
{
	int ignore_next = 0;
	int c;

	while((c = getchar()) != EOF)
	{
		if(c == ' ' && ignore_next)
			continue;
		else if(c != ' ')
			ignore_next = 0;
		else
			ignore_next = 1;
		putchar(c);
	}
	return 0;
}


vi ex1-9.txt 
This is  a   file    with
spaces     all      over!

cc ex1-9.c 
./a.out < ex1-9.txt 
This is a file with
spaces all over!
```
